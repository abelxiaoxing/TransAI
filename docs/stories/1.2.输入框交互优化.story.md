[Source: architecture/input-optimization.md#GTextEdit 组件重构]

### API Specifications
**需要实现的点击处理 API：**
```qml
MouseArea {
    anchors.fill: parent
    propagateComposedEvents: true

    onClicked: {
        if (mouse.button === Qt.LeftButton) {
            var pos = textEdit.positionAt(mouse.x, mouse.y + flickable.contentY);
            textEdit.forceActiveFocus();
            textEdit.cursorPosition = pos;
            mouse.accepted = true;
        } else {
            mouse.accepted = false;
        }
    }
}
```
[Source: architecture/input-optimization.md#GTextEdit 组件重构]

**TextEdit 现有属性：**
- `positionAt(x, y)` - 坐标到文本位置转换
- `forceActiveFocus()` - 强制获取焦点
- `cursorPosition` - 光标位置设置
[Source: Qt/QML TextEdit Documentation]
