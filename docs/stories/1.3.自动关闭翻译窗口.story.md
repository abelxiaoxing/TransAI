# Story 1.3: 自动关闭翻译窗口

## Status

**Status**: Done

## Story

**As a** 阅读翻译结果的用户，
**I want** 点击翻译窗口外部区域时窗口自动关闭，
**so that** 我的阅读流程不被中断，体验更加连贯。

## Acceptance Criteria

1. 翻译窗口支持点击外部区域自动关闭
2. 关闭动画流畅自然
3. 窗口位置计算避免遮挡重要内容
4. 不影响现有快捷键和系统托盘功能

### API Specifications
**需要实现的外部点击检测 API：**
```qml
MouseArea {
    anchors.fill: parent
    propagateComposedEvents: true

    onClicked: {
        if (mouse.button === Qt.LeftButton && !isPointInsideChildElements(mouse.x, mouse.y)) {
            closeWindowAnimation.start();
        }
    }
}
```

**窗口关闭动画 API：**
```qml
SequentialAnimation {
    id: closeWindowAnimation

    NumberAnimation {
        target: mainWindow
        property: "opacity"
        to: 0
        duration: 250
        easing.type: Easing.OutQuad
    }

    ScriptAction {
        script: {
            mainWindow.visible = false;
            mainWindow.opacity = 1; // 重置透明度
        }
    }
}
```
[Source: Qt/QML Window and Animation Documentation]

**窗口位置计算 API：**
- `Screen.width` 和 `Screen.height` 获取屏幕尺寸
- `mainWindow.x`, `mainWindow.y` 控制窗口位置
- `mainWindow.width`, `mainWindow.height` 获取窗口尺寸
[Source: Qt/QML Screen and Window Documentation]

**兼容性保证：**

**事件处理流程：**
```
用户点击外部区域 → MouseArea 检测 → 触发关闭动画 → 隐藏窗口 → 重置状态
```

- 动画时长控制在 200-300ms 之间

- 确保事件处理不会影响主线程性能

**跨平台兼容性：**
- 支持 Windows、macOS、Linux
- 考虑不同平台的窗口管理行为差异
- 确保系统托盘在所有平台上正常工作
